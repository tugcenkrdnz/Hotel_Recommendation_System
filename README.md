# Smart Hotel Finder

An intelligent search engine that recommends hotels based on user intent (e.g., "business trip", "romantic holiday") rather than just keywords.

## 🌟 Key Features
- **Semantic Search:** Uses `Cosine Similarity` to match user descriptions with hotel profiles.
- **Location Aware:** Dynamically filters and analyzes hotels based on the selected country.
- **Efficient Processing:** Uses `TF-IDF` to handle large-scale text data.

## 🚀 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
